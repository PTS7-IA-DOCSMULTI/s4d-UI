# s4d-UI
Interface for s4d, ENSIM 4A Project

# Install

## Clone
```
Clone s4d-UI
Copy requirements.txt and brain.py from communicationPythonElectronFlask 
```
## Install dependencies
```
pip install -r requirements
npm install
```


# Launch

### Launch python server
```
    python3 brain.py
```

### Launch app
```
    npm start
```
